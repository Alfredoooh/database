Política de Privacidade

Bem-vindo ao nosso app! A sua privacidade é muito importante para nós. Esta política descreve como coletamos, usamos e protegemos os dados dos usuários que utilizam nosso dashboard para análise de contas TikTok.

1. Dados coletados

Nosso app pode coletar os seguintes tipos de dados, apenas quando o usuário autoriza via TikTok Login:

  • Informações do perfil TikTok (nome de usuário, foto, bio, seguidores, seguindo, likes totais).
  • Dados de vídeos da conta do usuário (likes, comentários, visualizações, compartilhamentos).
  • Dados de desempenho de postagens quando o usuário utiliza a função de publicação.

2. Uso dos dados

Os dados coletados são usados exclusivamente para:

  • Analisar crescimento e desempenho da conta TikTok do usuário.
  • Exibir métricas e gráficos no dashboard para monitoramento de engajamento.
  • Facilitar a publicação de vídeos diretamente pelo app, se autorizado.
  • Melhorar funcionalidades do app e personalizar a experiência do usuário.

3. Compartilhamento de dados

Não compartilhamos os dados do usuário com terceiros, exceto quando necessário para:

  • Conectar com a API oficial do TikTok para autenticação e postagem.
  • Cumprir obrigações legais, como requisições de autoridades competentes.

4. Segurança

Adotamos medidas de segurança para proteger os dados do usuário, incluindo criptografia durante a transmissão e armazenamento seguro no servidor.

5. Consentimento

Ao utilizar o nosso app, o usuário concorda com a coleta e uso dos dados conforme descrito nesta política de privacidade. O usuário pode revogar o acesso a qualquer momento via TikTok Login.

6. Contato

Se tiver dúvidas sobre esta política de privacidade, entre em contato conosco pelo email: contato@seudominio.com
