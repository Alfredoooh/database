Teste AdSense

Este ficheiro verifica se o script AdSense carrega e se o bloco de anúncio responde. O anúncio abaixo está configurado para adtest=on (anúncio de teste).

Notas rápidas

  • Quando fizeres deploy, coloca o ads.txt (ve abaixo) na mesma pasta pública (raiz) para que fique acessível em https://SEU-DOMÍNIO/ads.txt.
  • Substitui data-ad-slot="1234567890" pelo teu ID real antes de passar para produção.
  • Se tudo estiver correcto, testes com data-adtest="on" mostram anúncios de teste; remove o atributo para anúncios reais.
